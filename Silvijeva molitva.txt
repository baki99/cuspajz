SUBINTERFACEOVI:
int fa 0/0.[broj]
enc dot1q [broj vlana]
ip add [IP] [subnet mask]
+ glavni interface ne smije imati IP i mora biti upaljen (no shut)
----------------------

TRUNK (u interfaceu):
switchport mode trunk

ACCESS (u interfaceu):
switchport mode access
switchport access vlan [broj vlana]

PROVJERA:
show int trunk
show vlan brief
